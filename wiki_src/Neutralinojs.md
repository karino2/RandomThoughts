## hello world

[Your First Neutralinojs App - Neutralinojs](https://neutralino.js.org/docs/getting-started/your-first-neutralinojs-app)　をやってみる。

```
$ npm install -g @neutralinojs/neu
$ neu create neumyapp
$ cd neumyapp
$ neu run
```

で起動した。あっさり起動するのは好印象。
