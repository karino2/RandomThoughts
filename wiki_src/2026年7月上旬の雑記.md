```js
/**
  現在のバッファの名前のbasenameをWikiLinkとしてコピー
*/
function wiki() {
  let buf = selected_buffer();
  let url = buf.url;

  if(buf.url) {
    let name = String(buf.name).replace(/\.md$/, "");
    copy_to_clipboard(`[[${name}]]`);
  }
}
```

buf.nameがJavaのStringになっちゃってJSのにならないのはちょっとかっこわるいが。
